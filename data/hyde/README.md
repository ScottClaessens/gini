# Information on dataset `hyde.csv`

**File name:** `hyde.csv`

**Title:** Time-series estimates of cropland areas and population sizes from the HYDE v3.5 dataset for 972 longitude-latitude coordinates

**Authors:** Kees Klein Goldewijk

**Reference:** Klein Goldewijk, K., Beusen, A., Doelman, J., & van der Linden, M. (2026). Holocene anthropogenic land-use reconstruction HYDE 3.5, Data in Brief, submitted.

**Source data URL:** <https://public.yoda.uu.nl/geo/UU01/F45D44.html>

**HYDE project URL:** <https://landuse.sites.uu.nl/hyde-project/>

**Date accessed:** 22nd June 2026

**Dimensions:** 80,676 rows x 5 columns

**Notes:** The `hyde.csv` dataset is generated by the R script `download_hyde_data.R`. This script loops over time slices in the HYDE dataset (-10,000 BC to 1980) and extracts time-series estimates of cropland areas and population sizes based on longitude-latitude coordinates for archaeological sites in the Gini dataset. The resulting dataset is in long format, with 80,676 rows (972 unique longitude-latitude coordinates multiplied by 83 time slices). See `download_hyde_data.R` for more details.

**Data dictionary:**

- `date` - dated year of the estimate (negative = BC, positive = AD), for linking to the Gini dataset
- `longitude` - longitude in decimal degrees, for linking to the Gini dataset
- `latitude` - latitude in decimal degrees, for linking to the Gini dataset
- `cropland` - total cropland area in km2 per grid cell
- `pop_size` - total number of inhabitants per grid cell
