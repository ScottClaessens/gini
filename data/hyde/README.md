# Information on dataset `hyde.csv`

**File name:** `hyde.csv`

**Title:** Estimates of cropland areas and population sizes for 1173 archaeological sites extracted from the HYDE v3.5 dataset

**Authors:** Kees Klein Goldewijk

**Reference:** Klein Goldewijk, K., Beusen, A., Doelman, J., & van der Linden, M. (2026). Holocene anthropogenic land-use reconstruction HYDE 3.5, Data in Brief, submitted.

**Source data URL:** <https://public.yoda.uu.nl/geo/UU01/F45D44.html>

**HYDE project URL:** <https://landuse.sites.uu.nl/hyde-project/>

**Date accessed:** 22nd June 2026

**Dimensions:** 1173 rows x 6 columns

**Notes:** The `hyde.csv` dataset is generated by the R script `download_hyde_data.R`. This script loops over the time slices in the HYDE dataset and extracts estimates of cropland areas and population sizes based on longitude-latitude coordinates and dates for archaeological sites in the Gini dataset. See `download_hyde_data.R` for more details.

**Data dictionary:**

- `site` - the name of the archaeological site, for linking to the Gini dataset
- `longitude` - longitude in decimal degrees, for linking to the Gini dataset
- `latitude` - latitude in decimal degrees, for linking to the Gini dataset
- `date` - dated year of archaeological site (negative = BC, positive = AD), for linking to the Gini dataset
- `cropland` - total cropland area in km2 per grid cell (average within 5km radius)
- `pop_size` - total number of inhabitants per grid cell (average within 5km radius)
